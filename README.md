# AccountWriter
우리 환경 (삼촌네 회사) 정산서 프로그램 제작

## 주제
정산서 프로그램 제작

## 핵심 기능
회계 파일을 기준으로 내부의 데이터들을 추출해 정해진 양식의 정산서 프로그램 제작
